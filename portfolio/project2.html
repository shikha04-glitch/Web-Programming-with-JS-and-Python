Recipe Haven

A web-based recipe sharing platform where users can create, share, and discover recipes easily. Recipe Haven is designed to offer a clean interface, smooth navigation, and an engaging user experience for food enthusiasts. Features include user authentication, recipe submission, search and filter options, user profiles, and a rating system.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • Node.js
  • Express.js
  • MongoDB
  • SQL

What I Learned

Improved understanding of web development concepts, user authentication, and RESTful API design.

Features

  • User authentication (signup/login)
  • Recipe submission and management
  • Search and filter recipes by category, ingredients, and ratings
  • User profiles with recipe history
  • Rating and review system for recipes
